# AList
Gestionnaire d'anime/manga avec d'autres fonctionnalités

## Dépendances :
- Python 3.5+
- Pillow 
- Jikanpy
- requests
- urllib3

## Install :

### PIP

- Launch in cmd : pip install AList
- launch in cmd : alist
- Enjoy

### Git

- Clone this repo
- Install requirements (pip install -r requirements.txt)
- Launch in cmd : python alist/AList.py
- Enjoy
